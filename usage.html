Tweetledee

A PHP library that provides an incredibly easy way to access Twitter data formatted as JSON or a RSS feed

View project onGitHub

Usage

The following examples demonstrate how to use Tweetledee on your web server. If you are having problems with Tweetledee, you can find more information about how to get help and report bugs here.

RSS Feed Types

User Timeline RSS Feed userrss.php

Generic User Timeline URL

  http://[yourdomain]/tweetledee/userrss.php
Optional URL Parameters
ParameterDescriptionDefaultValues
cTweet count251 - 200
userTwitter useryouany username
xrpExclude replies0 (=false)1 (=true)
xrtExclude retweets0 (=false)1 (=true)
User Timeline with 100 Tweets

  http://[yourdomain]/tweetledee/userrss.php?c=100
User Timeline of a Different User (@pixsied)

  http://[yourdomain]/tweetledee/userrss.php?user=pixsied
User Timeline, Excluding All Replies

  http://[yourdomain]/tweetledee/userrss.php?xrp=1
User Timeline, Excluding All Retweets

  http://[yourdomain]/tweetledee/userrss.php?xrt=1
All Options

  http://[yourdomain]/tweetledee/userrss.php?c=100&user=pixsied&xrp=1&xrt=1

Home Timeline RSS Feed homerss.php

Generic Home Timeline URL

  http://[yourdomain]/tweetledee/homerss.php
Optional URL Parameters
ParameterDescriptionDefaultValues
cTweet count251 - 200
xrpExclude replies0 (=false)1 (=true)
Home Timeline with 100 Tweets

  http://[yourdomain]/tweetledee/homerss.php?c=100
Home Timeline, Exclude Replies

  http://[yourdomain]/tweetledee/homerss.php?xrp=1
All Options

  http://[yourdomain]/tweetledee/homerss.php?c=100&xrp=1

Tweet Search RSS Feed searchrss.php

Generic Tweet Search Feed URL

  http://[yourdomain]/tweetledee/searchrss.php?q=[search_term]
Optional URL Parameters
ParameterDescriptionDefaultValues
cTweet count251 - 200
rtResult typemixedpopular | recent
qSearch query(required)any search term
Search for "test"

  http://[yourdomain]/tweetledee/searchrss.php?q=test
Search for "test", Return 200 Results

  http://[yourdomain]/tweetledee/searchrss.php?q=test&c=200
Search for "test", Realtime Tweet Search (i.e. recent)

  http://[yourdomain]/tweetledee/searchrss.php?q=test&rt=recent
Search for "test", Popular Tweet Search (i.e. popular)

  http://[yourdomain]/tweetledee/searchrss.php?q=test&rt=popular
All Options

  http://[yourdomain]/tweetledee/searchrss.php?q=test&rt=recent&c=75
